# Contributing to Project NOETIKO

We welcome contributions from the Biophysics, QED, and Signal Processing communities.

## How to Contribute
1.  **Fork** the repository.
2.  **Clone** your fork locally.
3.  Create a feature branch (`git checkout -b feature/AmazingFeature`).
4.  **Test** your changes using our CI pipeline (`python -m unittest tests/test_noetiko_physics.py`).
5.  Commit your changes.
6.  Open a **Pull Request**.

## Research Standards
All code submissions must adhere to the **v2.0 Simulation Constraints** defined in the README. Mathematical models must be validated against the derivations in `docs/MATHEMATICS.md`.
